git references
Command Description Flags / Args Example
init Initializes a new repository   git init
clone Clone a repository into a new directory   git clone "URL"
add Stage a file contents for commit
  • -n Dry Run: shows which files will be added
  • -u Update: only files currently tracked
  • -A All: all files in the project
 git add file.html
reset Clear staged files   git reset
commit Record staged changes to the repository -m <msg> Message: use the given <msg> as the commit message git commit -m"commit message"
revert Revert to a previous commit <commit> git revert commit
push Update remote repository with committed files
  • origin <branch name>
  • origin <tag>
  • --tags push all tags
  • : <tag> delete tag
 git push origin branch
fetch Download objects and refs from remote repository   git fetch origin branch
merge Join two or more development histories together -m <msg> Message: use the given <msg> as the commit message git merge
pull Does git fetch and merge from a remote repository   git pull origin branch
log Show commit logs   git log
tag Tag the last commit or specify a commit to be tagged <tag name>[<commit>] git tag tagName
diff Show changes from last commit [<path>] git diff file
branch Create a new branch <branch name> git branch name
checkout checkout a branch or commit
  • <commit or tag or branch>
  • -b <branch name> creates a new branch and checks it out
 git checkout branch name
